Prompt 1:

Create a clean, modern, responsive UI layout for a typing speed test web application. The layout should include: a header, a section for the random sentence, a typing area, a timer, a “Finish Test” button, and a results section. Use semantic HTML5 and responsive CSS with Flexbox or Grid. Provide clean and readable code.
